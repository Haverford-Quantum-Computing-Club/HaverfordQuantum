# Resources Folder

This folder contains all downloadable files for the Qiskit Fall Fest hackathon.

## Folder Structure

- `notebooks/` - Jupyter notebook files (.ipynb)
- `pdfs/` - PDF documents and reference materials
- `other/` - Other downloadable files

## How to Add New Files

1. Upload your files to the appropriate folder
2. Update the `resources-data.json` file in the root directory with the file information
3. Commit and push your changes to GitHub
4. The files will automatically appear on the Resources page

## File Naming Convention

Please use descriptive names with underscores:
- `day1_introduction_to_quantum_computing.ipynb`
- `quantum_gates_reference.pdf`
- `challenge_1_grover_algorithm.ipynb`
